ย้ายข้อมูล Google Sheets → Supabase

เครื่องมือนี้อ่านทุกตารางจาก Google Sheet (gviz CSV) แล้ว upsert เข้า Supabase ครั้งเดียว — ปลอดภัยต่อการรันซ้ำ (upsert by id).

⚠ ทำ ก่อน เปลี่ยน BACKEND เป็น 'supabase': (1) สร้าง project + รัน supabase/schema.sql แล้ว (2) กรอก SUPABASE_URL + SUPABASE_ANON_KEY ใน app/config.js (RLS ปิดอยู่ → anon key เขียนได้)
