ตั้งค่า Supabase Auth — สร้าง / เพิ่มผู้ใช้ + ตั้งรหัสเอง

สร้าง/อัปเดต/เพิ่มบัญชี Supabase Auth — พิมพ์รหัสเองได้ · เพิ่มคนใหม่ได้ในตัว · ฝัง role ใน app_metadata (ใช้กับ RLS)

🔐 service_role key คือกุญแจระดับสูงสุด — วางในหน้านี้ใช้ครั้งเดียวในเครื่องคุณเท่านั้น (อยู่ในหน่วยความจำ ไม่บันทึก ไม่ขึ้น repo). เอาจาก Supabase → Project Settings → API → service_role · secret (กด Reveal). ทำเสร็จแนะนำหมุน (regenerate) key ได้ (แอปใช้แค่ anon key).

ดึงรายชื่อจริงทั้งหมด รวม คนที่เพิ่มไปก่อนหน้า ที่ยังไม่อยู่ใน config.js (ไม่งั้นจะแก้รหัสเขาไม่ได้)

ทำชื่อผู้ใช้ (username)roleชื่อแสดงรหัสผ่านสถานะ

สรุปรหัส — ก็อป/ดาวน์โหลดเก็บก่อนปิดแท็บ (ดูย้อนหลังไม่ได้)
